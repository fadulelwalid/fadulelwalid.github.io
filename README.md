# 🌐 Personal Portfolio

A modern, responsive React-based personal portfolio showcasing background, selected projects, skills, and personal journey.

### 🔗 Live Site

[https://fadulelwalid.github.io](https://fadulelwalid.github.io)

---

### 🛠️ Tech Stack

* **Framework:** React (Create React App)
* **Language:** JavaScript
* **Styling:** CSS
* **Icons:** Unicons
* **Contact Form (email):** EmailJS
* **Deployment:** GitHub Pages

---

### 🚀 Deployment

This portfolio is deployed using **GitHub Pages**.

The live site is served from the 'gh-pages' branch, which is automatically updated whenever i run 'npm run deploy'. The source code remains on the 'master' branch.

If you're interested in setting up a similar deployment:

* Explore the repository structure and the scripts in 'package.json'
* Read the official Create React App guide for deploying GitHub Pages: [Deploy a React app to GitHub Pages](https://create-react-app.dev/docs/deployment/#github-pages)

For more general information about GitHub Pages, see the [official GitHub Pages documentation](https://docs.github.com/en/pages).
